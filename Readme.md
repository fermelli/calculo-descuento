# Codigo de ayuda

## Dependencias

json-server

```bash
npm install -g json-server
```

o

```bash
yarn global add json-server
```

## Instalación

### Instalación de dependencias

```bash
npm install
```

o

```bash
yarn install
```

### Ejecución

```bash
npm run dev
```

o

```bash
yarn dev
```

#### Para los datos falsos

```bash
json-server --watch data.json
```
